Codex skill for AI-generated HTML slide decks

HTML Slides Editor

Turn a generated HTML presentation into a direct editing surface: change text in the preview, drag images onto existing visuals, crop in place, and save the result back to the deck.

Install the skill View on GitHub

Built for the last mile of slide making.

AI agents can draft the deck. Humans still need fast, tactile control over final wording, image swaps, and visual framing. This skill keeps that loop inside the HTML preview.

Step 01 Generate HTML slides

Use your agent to create a normal HTML presentation with DOM text and image assets.

Step 02 Enable editing

The skill injects a compact editor bar, runtime, auto-save snippet, and local save server.

Step 03 Edit in the preview

Type directly on rendered text, drop images onto existing visuals, crop them, and reload safely.

Text editing and image replacement, visibly.

The runtime is intentionally small: a top editor bar, a clear image drop state, and crop controls that stay inside the original image frame.

Install and run.

Use the repository as a skill source. Persistent edits require the local save server, because direct file:// previews cannot write back to disk.

python3 .codex/skills/html-slides-editor/scripts/slides_editor_switch.py enable --autosave path/to/index.html
node path/to/html-slides-editor-server.js 8765
